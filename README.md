# phantom-fe-techode

Application that allows simulating bus movements and enabling passengers to track their locations & movements. 


## Enter Directory

```
$ git clone https://github.com/atlp-rwanda/phantom-fe-techode.git 
```
```
$ cd phantom-fe-techode 
```

## install
```
$ npm i 
```

## Development

```
$ npm run dev
```

## Production

```
$ npm run build
```

## Start Application

```
$ npm start
```
